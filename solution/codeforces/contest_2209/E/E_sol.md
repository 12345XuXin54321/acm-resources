# E. A Trivial String Problem

## 题意

定义函数 $f(t)$ 为满足以下条件的最大的 $k$
* 有 $k$ 个字符串 $p_1, p_2, \cdots, p_k$ 都是 $t$ 的前缀，
且 $t = p_1 + p_2 + \cdots + p_k$ ，这里 $+$ 表示字符串连接。

有一个长度为 $n$ 的字符串 $s$

有 $q$ 次询问，提供两个整数 $l_i$ 和 $r_i$, 你需要计算
$$\sum_{j = l_i}^{r_i} f(s[l_i..j])$$

## 思路

由于 $q$ 较小，因此可以对于每个询问分别计算。

计算 $z$ 函数，从 $i = 0$ 遍历字符串。
设置一个栈，存匹配的位置。
结构是匹配开始的位置和匹配的长度。
$i = 0$ 时，存 $(0, len)$

对于每次遍历到的位置 $i$ ，
设当前栈顶为 $(k, z_k)$ 。当

* $k + z_k - 1 > i$ 时，说明当前 $i$ 在匹配的前缀中
* $k + z_k$ 等于 $i$ 且 $z_i$ 不为 0 时，
说明当前 $i$ 的匹配的前缀刚好可以连接上一个匹配到的
* 其它情况说明无法刚好连接，需要弹出栈顶

当 $z_i$ 不为 0 时，可以把当前位置入栈。

答案为所有位置的栈深的和。
