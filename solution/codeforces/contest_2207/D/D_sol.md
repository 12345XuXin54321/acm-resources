# D. Boxed Like a Fish

## 题意

有一个正整数 $k$ 和一棵有 $n$ 个节点的树。

Cyndaquil 正在穿越这棵树并尝试到达一个叶子节点。
开始时他在一个非叶子节点，每个回合可以选择待在原地，
或者选择一条边到达当前节点的邻居。

然而 Snorlax 尝试阻挡 Cyndaquil 。当 Snorlax 选择了一条边，
Cyndaquil 就不能经过了，直到 Snorlax 离开。此外，
同一时间只能禁止一条边，于是只有最近选择的边被禁止。

Snorlax 需要一些时间去选择一条新的边，他有一个冷却计时器，
开始为 0 ，他只有在计数器为 0 时才能选择一条新的边，
但不必立即完成。当他移动到新的边时，计时器会重置为 $k$ 。
在每个 Cyndaquil 的回合之后，计数器的值下降 1 。

Snorlax 先行动，开始时他不在任何边，假设他们都执行最优的操作，
请问 Cyndaquil 在有限回合后是否能到达叶子节点？

## 思路

如果想去到其他节点，必须至少有两个选择，
因为封了一条边可以去另一条边。

获取每个节点离叶子节点的最小距离 $d_{min}$ ，
如果一个节点可以通过多个子节点去到叶子节点，
设其中两个最小距离为 $d_1, d_2$ ，
则如果满足 $d_1 + d_2 - 2 < k$ ，
从这个节点出发就一定可以去到叶子节点。
设这样的节点 $d_{min}$ 为 0 。

可以以 $v$ 为根节点进行 `dfs` 。
如果最后 $v$ 的 $d_{min}$ 为 0 则说明可以到达叶子节点。
