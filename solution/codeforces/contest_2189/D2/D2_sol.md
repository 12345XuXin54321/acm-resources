# D2. Little String (Hard Version)

## 题意

一个字符串 $w_1 w_2 \dots w_n$ 只包含字符 `0` 和 `1` 。定义函数 $f(w)$ 为 $[0, 1, \dots, n-1]$ 的排列 $p$ 中，满足以下条件的数量：

* 若 $w_i = 1$ ，则存在 $1 \leq l \leq r \leq n$ ，满足 $mex([p_l, p_{l+1}, \dots, p_r]) = i$
* 若 $w_i = 0$ ，则不存在这样的 $l$ 和 $r$

给定字符串 $s_1 s_2 \dots s_n$ ，和一个正整数 $c$ ；
考虑把 $s$ 中的所有字符 `?` 替换为 `0` 或 `1` 得到的所有 $w$ ，
计算 $f(w)$ 中，满足不能被 $c$ 整除的最小值。若不存在则输出 $-1$ 。
答案对 $10^9 + 7$ 取模。

## 思路

这个版本给定的字符串多了 `?` 。当 $w_i$ 为 `?` 时，相当于在 $\times 2$ 和 $\times i$ 中任选。当 $i \neq 1$ 或 $i$ 为偶数时，选择 $\times 2$ 一定会更优。可以把待确定的因数记录下来，这些因数都为奇数。

把 $c$ 的质因数中，已经确定会选择的删去，若全部删完了，就说明不存在。

剩下的质因数中若有除 $2$ 以外的质因数，则可以全部选择 $\times 2$ ；
否则必须至少剩下一个质因数 $2$ ，需要选择尽可能多的 $\times 2$ ，剩余部分选择尽可能小的 $\times i$ 。
