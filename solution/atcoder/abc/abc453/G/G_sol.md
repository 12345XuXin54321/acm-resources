# G - Copy Query

## 题意

有 $N$ 个长度为 $M$ 的序列 $A_1, A_2, \cdots, A_N$ 。
初始时，序列中的所有元素都为 0

序列 $A_i$ 的第 $j$ 个元素表示为 $A_{i,j}$

有以下三种操作：

* 把序列 $A_{X_i}$ 覆盖为 $A_{Y_i}$
* 把 $A_{X_i, Y_i}$ 改为 $Z_i$
* 计算序列 $A_{X_i}$ 的区间 $[L_i, R_i]$ 的和，并输出

## 思路

可以使用可持久化线段树来维护。
每个位置保存序列的某个版本。

操作 `1` 相当于整个序列 $A_Y$ 加上 $0$ ，
然后把版本保存到 $A_X$

操作 `2` 是单点修改序列 $A_X$ ，然后更新版本

操作 `3` 是计算序列 $A_X$ 的区间和
